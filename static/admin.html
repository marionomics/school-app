Como usar el sistema

  • 1. Crea una clase - Usa el boton "Crear Clase" para generar un codigo unico
  • 2. Comparte el codigo - Los estudiantes usan este codigo para inscribirse
  • 3. Gestiona tu clase - Haz clic en una clase para acceder a:
    • - Tomar asistencia
    • - Registrar calificaciones
    • - Aprobar participaciones
    • - Ver el progreso de cada estudiante
